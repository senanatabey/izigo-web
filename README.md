# IZIGO

Azərbaycan üçün turizm marketplace-i — Villalar, Avtomobillər, Transferlər və Tədbirlər.
Komissiyasız, ev sahibi ilə birbaşa WhatsApp əlaqəsi üzərində qurulub.

## Texnologiyalar

- React (Vite)
- React Router
- Supabase (planlaşdırılır — Postgres + Auth + Storage)

## İşə salmaq

```bash
npm install
npm run dev
```
